# RoboOrchard Build

**robo_orchard_build** is a set of tools designed to simplify the build process for the **RoboOrchard** project.

TBC

## Getting started

### Requirements

* Python 3.10

### Installation

#### From pip

```bash
pip install robo_orchard_build
```

#### From source

```bash
make install
```

## Contribution Guide

### Install by editable mode

```bash
make install-editable
```

### Install development requirements

```bash
make dev-env
```

### Lint

```bash
make check-lint
```

### Auto Format

```bash
make auto-format
```

### Type checking

```bash
pyright
```

### Run test

```bash
make test
```
